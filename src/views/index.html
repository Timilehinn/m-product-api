HOW IT WORKS

Base url: https://pacavel-m-api.herokuapp.com

  • POST baseurl/v1/user/register -- accepts req.body { email } to register
    response -> { message: String, status: Boolean }
  • POST baseurl/v1/user/login -- accepts req.body { email } to login, an email will sent will the url to login
    response -> { message: String, status: Boolean, token }, store token for recurrent authorisation
  • from the email -> baseurl/v1/user/verify-user-login?token=token
    response -> { message: String, status: Boolean } note: this will redirect to your domain
  • from the email -> baseurl/v1/user/verify-user-access -- accepts headers: { "x-access-token" : token } stored token
    response -> { message: String, status: Boolean }
    • contact the developer for more info `\_('_')_/`
